# AI-Powered Personal Developer Portfolio

A world-class, premium developer portfolio website designed and developed for **Inchara S Sathish**—positioning her as an AI & Machine Learning Enthusiast, Future Software Engineer, and Technology Explorer.

## 🚀 Deployed URL
The application is deployed live on Vercel:
**[View Live Portfolio](https://vercel.com/incharassathish77-6951s-projects/portfolio)**

---

## ✨ Features

- **Premium Dark Theme**: Implemented using high-contrast neon accents (cyan, purple, and blue) with frosted glassmorphism elements.
- **Dynamic Typing Animation**: Alternates through key personal branding statements in the Hero section.
- **Responsive Navigation**: Full mobile-friendly navigation layout with automatic active-link tracking.
- **Custom Cursor**: Interactive cursor circle and pointer tracking mouse coordinates.
- **HTML5 Particles Canvas**: Simulated math-based node network interacting in the background.
- **Skills Dashboard**: Categorized skill bars (Programming, AI/ML, Data Science, Web Tech, Soft Skills) with dynamic loading animations.
- **Interactive Projects Showcase**: Real-time project search and filtering capability.
- **GitHub Analytics**: Dynamic language breakdown chart and contribution heatmap grid showing contribution statistics.
- **AI Knowledge Hub**: Integrated Glossary terms, AIML learning roadmap, and technology trend trackers.
- **Simulated AI Chat Assistant Widget**: Interactive NLP-based chatbot responding to queries about resume highlights (Education, Projects, Skills, and Certifications).
- **Theme Toggle**: Fast switching between Light and Dark mode options.
- **Validated Contact Form**: Full visual validation handler for sending messages.

---

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Variables, Animations, Glassmorphism)
- **Logic**: ES6+ JavaScript
- **Icons**: FontAwesome v6
- **Hosting**: Vercel

## 💻 Running Locally

To run the portfolio website locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/incharassathish77-dotcom/Portfolio-.git
   cd Portfolio-
   ```

2. Start a local server (e.g., using Python):
   ```bash
   python -m http.server 8000
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```
